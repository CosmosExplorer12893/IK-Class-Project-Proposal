"Where in the World is My Next Vacation?"

A website designed to find where your personal paradise is in the world.

Team Members

Ian King

Application Overview

"Where in the World is My Next Vacation" is a program designed to find the user's personal paradise based on their specifications via their input. This program will analyze the user's answers to several questions, including, but not limited to, "what season of the year do you want to vacation?", "how long do you want to stay?", "do you want to go to a theme park?", "resort?", etc., and more.

Technology Overview

  • Languages: HTML5, CSS3, JavaScript (ES6+)
  • Libraries/Frameworks: Tailwind CSS, Chart.js
  • Platform: Developed in Visual Studio Code
  • Hosting: GitHub Pages
